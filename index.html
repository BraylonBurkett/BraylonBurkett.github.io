Links OS (2023)

Links 5 or "Los5" is a virtual operating system or secondary operating system made to run on even the lowest end of computers. I even personally installed it on my raspberry pi 3b (a really low end device the size of a credit card) and it ran as smooth as butter. It can quite literally run on any operating system created within the last 22 years! The point of Los5 is to give you a good user experience while your primary operating system does all the hard work. If you're interested you can check it out atThe official Links OS site


MAPLE (2023)

MAPLE is an acronym for Mathematical. Analytical. Processing. Language. Executable. MAPLE is a lightweight, portable coding language made to replace harder languages such as C++ or C. Despite the fact that MAPLE is a one-man-made coding language the first version was made within the span of 2 months(not rushed suprisingly). C++ was in mind when MAPLE was created, due to the language's efficiency and rhythmic nature. You can download the very first edition of MAPLE atThe official MAPLE siteThe website contains the MAPLE terminal to run your code, and the first edition of the MAPLE Guide. Happy Coding!
